# Power-BI-Performance-Dashboard 

## 📌 Introduction  
This project follows a portfolio dashboard tutorial and showcases my ability to analyze data and build an interactive analytics dashboard.

The primary goal was to transform raw data into meaningful insights using data cleaning, analysis, and visualization techniques.

It serves as both a portfolio project and a practical learning experience in business intelligence and data analytics.

---

## 📖 Overview  
The project involves cleaning, transforming, and analyzing a dataset before building an interactive dashboard to visualize key insights.

I worked on different types of analytics tasks such as:

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- KPI development and performance metrics  
- Trend analysis over time  
- Customer/Product segmentation  
- Building interactive visual dashboards  

This repository demonstrates how raw data can be transformed into actionable business insights using analytics and visualization tools.

---

## 🛠 Tools & Skills Used  

- Power BI  
- Data Modeling  
- DAX / Calculated Fields  
- Data Cleaning & Transformation  
- KPI Development  
- Data Visualization  

---

## 📚 What I Learned  

Through this project, I gained experience in:

- Cleaning and preparing messy datasets for analysis  
- Creating meaningful KPIs that reflect business performance  
- Designing interactive dashboards for better decision-making  
- Using calculated measures and advanced formulas  
- Presenting data insights in a clear and professional way  
- Thinking from a business perspective rather than just a technical one

## ⚡ Challenges I Faced  

### Data Cleaning  
Handling missing values, inconsistent formats, and ensuring the dataset was analysis-ready.

### Data Modeling  
Understanding relationships between tables and building an efficient data model.

### Creating Meaningful KPIs  
Choosing the right metrics that provide business value rather than just displaying numbers.

### Dashboard Design  
Balancing aesthetics with clarity to avoid clutter while keeping the dashboard informative.
